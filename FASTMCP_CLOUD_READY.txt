UPLOAD TO FASTMCP CLOUD:

1. Go to FastMCP cloud dashboard
2. Create new server project
3. Upload these files/folders:
   ├─ main.py
   ├─ config.py
   ├─ requirements.txt
   ├─ pyproject.toml
   ├─ README.md
   ├─ .env.example
   ├─ prompts/
   └─ resources/
4. Set environment variables:
   ├─ MCP_TRANSPORT=sse
   ├─ MCP_HOST=0.0.0.0
   └─ MCP_PORT=8000
5. Deploy!

═══════════════════════════════════════════════════════════════════════════════

FASTMCP CLOUD SETTINGS:

  MCP_TRANSPORT: sse               (FastMCP cloud uses SSE)
  MCP_HOST: 0.0.0.0               (Listen on all interfaces)
  MCP_PORT: 8000                  (Or platform-assigned)
  MCP_ENV: production
  ENABLE_LOGGING: false           (For performance)
  LOG_LEVEL: WARNING

═══════════════════════════════════════════════════════════════════════════════

AFTER UPLOAD:

You'll get a CloudServer URL like:
  https://api.fastmcp.xyz/servers/your-server-id

Use this to:
  1. Connect Claude Desktop (if supporting local connections)
  2. Call tools via API
  3. Scale automatically
  4. Monitor logs in FastMCP dashboard
